# Frames Creator

A browser-based tool for creating complementary split-frame Instagram posts (4:5 format).

**Live:** [silvanrapp.github.io/frames-creator](https://silvanrapp.github.io/frames-creator/)

## How it works

Upload four photos across two zones — top and bottom background, plus a small inset frame for each half. The result is a single 1080×1350 px image where two photos visually reference each other.

## Features

- Tap or drag & drop to load photos
- Pan and pinch/scroll to reposition and zoom each photo
- White or black frame border, with on/off toggle
- Export at 3× resolution (3240×4050 px)
- Works on iPhone and desktop
